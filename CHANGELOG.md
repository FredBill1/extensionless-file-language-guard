# Change Log

All notable changes to the "extensionless-file-language-guard" extension will be documented in this file.

## 0.0.1

- Adds the `Extensionless Plain Text` language and guard behavior for saved extensionless files.
- Allows a user-selected language to override the guard for the current open editor session.
- Adds settings for enabling the guard, allowed URI schemes, ignored basenames, and status messages.
